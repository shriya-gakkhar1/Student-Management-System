
# Student Management System (Java + JDBC)

## Features
- Add, View, Search, Update, Delete students
- JDBC + MySQL database
- DAO-Service architecture
- Exception handling

## Tech Stack
- Java
- JDBC
- MySQL

## How to Run
1. Add MySQL Connector/J to lib/
2. Compile:
   javac -cp "lib/mysql-connector-j-8.0.33.jar;." src/**/**/*.java
3. Run:
   java -cp "lib/mysql-connector-j-8.0.33.jar;." main.S
